Your DealFlow UI is eligible to be a **Verisense Mini App**.

## Steps to Register

1.  **Deploy the UI**: Ensure your Next.js app (`ui/`) is deployed to a public URL (e.g., Vercel, Netlify).
2.  **Go to Dashboard**: Visit [Verisense Dashboard](https://dashboard.verisense.network).
3.  **Register Mini App**:
    *   Click "Create Mini App".
    *   **Name**: DealFlow
    *   **Description**: Autonomous Sponsorship Swarm Dashboard.
    *   **URL**: Enter your deployed UI URL.
    *   **Icon**: Upload the project logo.
4.  **Connect Agent**:
    *   Link your Mini App to the A2A Agent you deployed.
    *   This allows the Mini App to send tasks to the agent directly.

## Integration

To make your UI fully integrated, you can use the Verisense React SDK (once available) to handle user authentication via their Verisense ID.

For now, your UI works as a standalone interface that can be embedded or linked from the Verisense ecosystem.
